# Product Scope

## 定位

Research Community Hub 是面向炒股社群成员的会员制投研内容网站，用于沉淀和展示社群已有的信息加工能力。

## V1 内容模块

1. **研报精选**：Hermes 将英文研报加工成中文网页内容，通过 API 上传。
2. **重要事件**：展示每周、每月值得关注的宏观、政策、产业和市场事件。
3. **宏观课程**：展示一套共 18 章的宏观经济基础课程。

## 用户与权限

- 访客：查看标题、摘要、免费试读和基础事件信息。
- 会员：查看完整研报、事件分析、全部会员课程和历史归档。
- 管理员：创建用户、设置会员期限、管理内容、配置审核模式。
- Hermes：持有 API Token，通过 `curl` 或程序提交内容和素材。

## 内容边界

网站负责：

- API 鉴权
- Schema 和业务规则校验
- HTML 安全清洗
- 幂等和重复检查
- 存储
- 人工审核或自动发布
- 会员权限控制
- 前台阅读展示
- 导入日志和错误返回

网站不负责：

- 英文 PDF 解析
- 翻译、总结和研报加工 Skill
- 使用 AI 补全缺失字段
- 修复 Hermes 上传的内容

当数据缺失或不合规时，接口返回机器可读的字段级错误，由 Hermes 修正后重试。

## 关键产品决策

- 一篇文章只保存一份完整 `body_html`。
- 通过 `<!-- PAYWALL -->` 区分公开试读和会员正文。
- 会员正文必须在服务端裁剪，不能完整下发后由前端隐藏。
- 人工审核支持全局开启或关闭。
- 审核关闭时，合法内容直接发布；不合法内容仍然拒绝。
- V1 不开放用户自主注册，由管理员创建用户名和初始密码。

## V1 非范围

- 在线支付和自动续费
- 微信登录、短信验证码
- 股票实时行情、选股、回测和组合管理
- 评论、发帖和私信
- 原始英文 PDF 下载
- APP 原生客户端
